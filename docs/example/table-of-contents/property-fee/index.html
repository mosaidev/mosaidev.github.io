    物业费 #

    APP在线缴物业费 #

    1. 用户在入住房间后，定期会有物业费账单
    2. 查看缴费账单详情，确认信息无误后缴费，选择支付方式，支付完成则缴费成功
    3. 在完成列表查看已支付的账单数据

    APP预存物业费 #

    1. 用户入住房间后，可以预缴物业费
    2. 在账单缴费页面有预缴物业费入口
    3. 确认预缴费房间、预缴金额，然后立即缴费，支付完成后则预缴物业费成功.
    4. 在预缴记录列表查看账单数据

    app预存后可以在线余额抵扣，系统也会在第2日自动抵扣 #

    管理员收取物业费 #

    1. 管理员在收付款列表查看未付款物业费
    2. 实际收费后，进入账单详情，新增流水数据
    3. 此时流水会根据账单默认回填数据，要填入支付方式和其他相关信息，无误后确认
    4. 流水与账单自动匹配后，物业费账单已付全款，该账单收取物业费结束。

    物业费设置 #

    1. 需要配置付款日、费用生成日、计费周期、费用生成策略
    2. 配置不同房源类型的单价
